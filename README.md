# LÖVE Platform Game

A simple 2D platform game created with LÖVE (Love2D).

## Requirements

- LÖVE 11.4 or newer (https://love2d.org)

## Running the Game

```bash
# From the project root
love src
```

## Controls

- Left Arrow: Move left
- Right Arrow: Move right
- Space: Jump

## Building

To create a .love file:
```bash
cd src
zip -9 -r ../game.love .
```
